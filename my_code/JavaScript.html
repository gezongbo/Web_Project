1.1 什么是JavaScript

JavaScript是一种基于对象和事件驱动的解释性脚本语言

JS 是一种网页编程技术，用来向HTML页面添加交互行为，由浏览器解析执行。

JS特点:
(1)
(2)JS由浏览器解析。
(3)JS内置许多对象。
(4)适用地方:浏览器端数据计算，表单的合法验证，事件触发，显示效果制作，异步数据提交。

1.4 数据类型

1.4.1 基本数据类型

  1. number 数字类型 数字(整数，小数)

1.4.2 特殊类型

  1. undefined undefined类型，变量声明未初始化，它的值就是undefined
  2. null null类型，表示空对象，如果定义的变量将来准备保存对象，可以将变量初始化为null,在页面上获取不到对象，返回的值就是null

1.4.3 复杂类型
