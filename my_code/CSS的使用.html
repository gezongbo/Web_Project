1. CSS的使用

1.1 CSS样式的使用方式

1.1.1 内联样式

写在开始样式中，针对单独的一个标记元素起作用

style=""属性写在开始标记中

  1. h1 标题标记
  2. div 块容器标记
  3. p 段落标记

1.1.2 内部样式

写在HTML页面中的head标记里

1.1.3 外部样式

写在单独定义的一个以css结尾的文件中，该文件要引入

1.2 CSS样式的优先级

  1. 内联样式
  2. 外部样式或者内部样式 就近原则
  3. 浏览器默认设置样式

1.3 选择器

1.3.1 元素选择器/标记选择器

1.3.2 类 class 选择器

常规用法:
先在开始标记中定义class=""属性，选择器通过className()的方式调用
高级用法:将标记选择器和类选择器结合使用
语法:标记选择器.className{}

1.3.3 id 选择器

根据id的值来查找,id保证唯一
选择器通过#idName()的方式来获取

1.3.4 选择器分组

一般将一些相同的样式作用于多个标记元素中

1.3.5 派生选择器

依据标记元素在其上下文关系定义样式

1.3.6 伪类选择器

针对某些标记添加特殊的效果
语法:使用冒号作为分隔符，左边是选择器，右边是伪类;
:link 未访问
:visited 访问后
:hover 鼠标指向时
:active 鼠标按下时

1.4 尺寸和边框

1.4.1 尺寸/大小

默认显示，即使超出尺寸，也显示全部内容；
overflow:当内容溢出时如何处理
visible 默认
hidden 超出部分隐藏
scroll 显示滚动条
auto 自动适应

1.4.2 边框

(1)简单写法:border:width style color; 分别表示:线宽 样式 颜色；
(2)单边写法：border-left/right/top/bottom:width style color
(3)只设置边框的线宽:border-width
(4)只设置边框的类型:border-style
(5)只设置边框的颜色:border-color

1.5 框模型-盒子模型(包括:外边框+边框+内边框)

1.5.1 内边距:内容与边框之间的距离 padding:value

(1)简单写法:padding:v1 v2 v3 v4 (分别表示:top right bottom left)
(2)单边写法:padding-left/right/top/bottom:value

1.5.2 外边距:与下一控件之间的距离

(1)简单写法：margin: v1 v2 v3 v4 (分别表示:top right bottom left)
(2)单边写法:margin-left/right/top/bottom:value

1.6 背景

1.6.1 背景相关样式

(1)背景颜色

background-color:red;

(2)背景图片

background-image:url("image\\1.jpg");

NULL

(3)背景图片重复问题(可以使用background-repeat属性控制背景图片的平铺效果)

background-repeat:repeat 默认情况下，背景图片在水平和垂直方向上重复出现，创建一种称为"墙纸"的效果
background-repeat:repeat-x 仅在水平方向上重复
background-repeat:repeat-y 仅在垂直方向上重复
background-repeat:no-repeat 仅显示一次

(4)背景图片的位置

background-position:x y 分别表示水平、垂直位置
background-position:left、center、right、top、bottom 分别表示左边，居中，右边，上边、下边显示。

(5)背景图片的固定

background-attachment:scroll 默认，背景图片会随着页面滚动而移动。
background-attachment:fixed 固定

1.8 定位

1.8.1 普通流定位

我是p标记

我是div标记
我是span标记 我是超链接

1.8.2 浮动定位

(1)将元素排列在普通流定位之外

(2)将元素浮动以后，可以向左边或者右边移动，直到碰到边框或其他浮动元素为止

(3)浮动定位一般实现特定的定位效果

设置浮动float:left/center/right
清除浮动clear:left/center/right

1.8.3 相对定位，绝对定位，固定定位

(1)position属性:position:relative;更改定位模式为相对定位
(2)偏移属性:top/bottom/left/right:value;
(3)堆叠顺序:z-index:value;

一旦修改了元素的定位方式，则元素可能会发送堆叠。
可以使用z-index属性来控制元素出现堆叠的顺序
z-index属性：值为数值，数值越大表示堆叠顺序更高，即距离用户更近

1.8.4 绝对定位

position属性:position:absolute;更改定位模式为绝对定位

对元素进行绝对定位的设置，会让该元素从普通定位流中移除，不占用空间

1.8.5 固定定位

position属性:position:fixed;更改定位模式为固定定位

特点:
(1)固定定位会将元素从普通流定位中完全移除，不占用页面的空间
