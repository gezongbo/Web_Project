1.5 表单

表单的作用:用于显示，收集信息，并提交信息到服务器端，服务器接收到信息后，再保存到数据库端。

表单项的分类

(1)input标记系列

  1. 用户名输入框 text
  2. 密码输入框 password
  3. 单选 radio 不同选项有相同name
  4. 多选 checkbox
  5. 提交按钮 submit
  6. 重置按钮 reset
  7. 普通按钮 button
  8. 隐藏域 hidden 不给用户看到
  9. 文本选择框 file 上传文件

(2)非input标记系列

  1. 文本标记 label
  2. 下拉框(选项框) select
  3. 多行文本框 textarea
  4. 控件分组 fieldset

(3)action 指定表单提交服务器端的地址

(4)method=“get/post"
  get方式,默认，将表单的数据通过浏览器地址栏的方式进行提交(快，不安全)。
  post方式,将表单的数据以协议体的方式提交。
